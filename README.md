# unit-converter

This is a unit converter web app, idea is from [roadmap.sh](https://roadmap.sh/projects/unit-converter)

This is very easy to use

If you want to install locally

Copy, paste this below line in terminal on vs code and navigate to the folder in terminal

git clone https://github.com/caals666/unit-converter

cd unit-converter

npm install

nodemon index.js

It will run on http://localhost:3000/
